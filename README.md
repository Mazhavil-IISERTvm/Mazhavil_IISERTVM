# Mazhavil Club Email Assets

This repository serves as a permanent storage host for image assets used in previous HTML newsletters and emails sent out by the **Mazhavil Club of IISER Thiruvananthapuram**. 

---

## CRITICAL WARNING: DO NOT MODIFY

> **STOP! Read this before making any changes.**
> 
> Because older sent emails reference the exact URLs of the files in this repository, any modifications will permanently break the images in those emails.

*   **DO NOT DELETE** any files or folders.
*   **DO NOT REARRANGE** or move files into different directories.
*   **DO NOT RENAME** this repository.
*   **DO NOT RENAME** any existing images or assets.

---

## 📂 Purpose & Usage

* **Why is this strict?** HTML emails rely on absolute URLs to display images. If a file path or the repository name changes, the link breaks, leaving past emails looking broken to the recipients.
* **Adding new assets:** If you need to host images for *new* emails, you may upload them here, but **never** alter or overwrite what is already present.
